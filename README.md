# 🃏 BlackJack

Jogo de cartas BlackJack (21) desenvolvido em Java com interface gráfica em JavaFX.

## 🛠️ Tecnologias

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![JavaFX](https://img.shields.io/badge/JavaFX-007396?style=for-the-badge&logo=java&logoColor=white)

## 🎮 Funcionalidades

- Lógica completa do BlackJack com as regras clássicas do jogo
- Interface gráfica com JavaFX
- Jogador versus dealer com lógica automática para o dealer
- Controle de cartas, pontuação e condições de vitória/derrota/empate

## ▶️ Como executar

### Pré-requisitos

- Java 11+
- JavaFX SDK

### 1. Clonar o repositório

```bash
git clone https://github.com/ErkPatrick/blackJack.git
cd blackJack
```

### 2. Executar

Importe o projeto em uma IDE com suporte a JavaFX (Eclipse ou IntelliJ) e execute a classe principal.

Ou via terminal com JavaFX configurado:

```bash
java --module-path /caminho/para/javafx/lib --add-modules javafx.controls,javafx.fxml -cp bin Main
```
